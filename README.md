# Logistic Regression – Airline Customer Satisfaction Prediction

## Project Overview
This project builds a predictive binary classification model using Python and Scikit-learn to analyze airline passenger survey data. By leveraging passenger demographics and flight experience ratings, we implement a Logistic Regression framework to compute the mathematical probability of customer satisfaction. These metrics allow airlines to pinpoint key friction points in the passenger lifecycle and engineer proactive customer retention frameworks.
